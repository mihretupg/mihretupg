# Mihretu Petros Portfolio

A professional portfolio website for **Mihretu Petros**, a Virtual Assistant, Executive Assistant, Administrative Assistant, E-commerce Support Specialist, and Frontend Web Developer based in Addis Ababa, Ethiopia.

The site presents Mihretu's professional background, remote work experience, technical skills, testimonials, training, and contact information in a polished, responsive single-page portfolio.

## Overview

This portfolio is designed to help clients, employers, and collaborators quickly understand Mihretu's strengths across virtual assistance, e-commerce operations, administrative support, website management, and frontend development.

It highlights practical experience with remote workflows, online store administration, customer communication, content updates, scheduling, project coordination, and modern web tools.

## Key Features

- Responsive single-page portfolio built for desktop, tablet, and mobile
- Professional hero section with clear role positioning
- Experience section covering virtual assistant, e-commerce, and website support work
- Skills grouped by virtual assistance, e-commerce, development, marketing support, and tools
- Testimonials and references from previous collaborators
- Education, training, language, and contact sections
- Downloadable CV asset
- Vercel-friendly routing configuration

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

The project is ready to deploy on Vercel or any static hosting provider that supports Vite applications.

For Vercel, the included `vercel.json` routes all requests to `index.html`, which supports the single-page application experience.
